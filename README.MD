# Cube of Iron

## Basic Movement
- *ONLY FORWARD MOVEMENT*
- **Space** - Jump  
- **Shift** - Dash forward  
- **Mouse Movement** - Look around  

### Camera Control
- **Move Mouse** - Control camera view  
- **Middle Mouse Button** - Quick 180Â° turn  

## Combat System

### Sword Attacks
- **Left Mouse Button** - Basic swing attack  
  - First click: Swing 1  
  - Follow-up within 1 second: Swing 2  
  - Enables hitbox during animation  

### Special Abilities (Hand Attacks)
- **Right Mouse Button** - Cast selected special ability  
  - 5-second cooldown between uses  
  - Casts after the `hand_use` animation completes  

### Ability Selection
- **Q** - Select Fireball (currently the only implemented ability)  
- **W** - Placeholder ability (unimplemented)  
- **E** - Placeholder ability (unimplemented)  
- **R** - Placeholder ability (unimplemented)  

## Current Features
- Mouse-locked first-person camera  
- Sword combat with two-swing combo system  
- Fireball projectile attack  
- Dash ability for quick movement  
- Invincibility frames after taking damage
